!1 Attribut einer Gruppe ändern
Wird im eDir ein im Treiberfilter entsprechend gelistetes Attribut geändert
|script|Ldap Operation Script|metatree|
|add attribute| description | with value | ein paar infos | to Object | ${scoped_meta_group} |

|Sleep|3000|

Wird diese Änderung ins AD übertragen
|script|Ldap Operation Script|ad|
|ensure| attribute | description | contains value | ein paar infos | for object | ${scoped_ad_group} |

!1 User in Scope Gruppe aufnehmen
Wird ein User Mitglied einer Gruppe
|script|Ldap Operation Script|metatree|
|add user | ${metauser} | to group | ${scoped_meta_group} |
|add user | ${metauser} | to group | ${unscoped_meta_group} |

|Sleep|8000|

Wird die Gruppenmitgliedschaft für snychjronisiert Gruppen (scoped) übermittelt
|script|Ldap Operation Script|ad|
|ensure| attribute | member | contains value | ${aduser} | for object | ${scoped_ad_group} |
|reject| attribute | memberOf | contains value | ${unscoped_ad_group} | for object | ${aduser} |




!1 User aus Scope Gruppe ausschliessen
Wird einem User eine Gruppenmitgliedschaft entzogen
!!|script|Ldap Operation Script|metatree|
!!|remove user | ${metauser} | from group | ${scoped_meta_group} |

!!|Sleep|3000|

wird dies auch ins AD synchronsiert
!!|script|Ldap Operation Script|ad|
!!|reject| attribute | Member | contains value | ${aduser} | for object | ${scoped_ad_group} |

!1 Gruppe umbennen

Wird im eDir eine Gruppe umbenannt
|script|Ldap Operation Script|metatree|
|rename entry| ${scoped_meta_group} | to | cn=apple,${meta_grp_ou}|

| Sleep|3000|

spiegelt sich das auch in der AD Gruppe wieder
|script|Ldap Operation Script|ad|
|ensure | object |CN=apple,${ad_grp_ou}|exists|

(Aufräumen)
|script|Ldap Operation Script|metatree|
|rename entry| cn=apple,${meta_grp_ou} | to | ${scoped_meta_group}|
