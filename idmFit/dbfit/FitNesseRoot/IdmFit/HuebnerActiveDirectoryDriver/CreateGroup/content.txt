!1 Erzeugen einer Gruppe im eDirectory AD Gruppen Container
Im eDirectory wird eine Gruppe im Container ${meta_grp_ou} erzeugt, 

|script|Ldap Operation Script|metatree|
|create object|${scoped_meta_group}|with class | groupOfNames| and attributes | !{description: Scoped Group}|

die nach kurzer Zeit im Active Directory im Container ${ad_grp_ou} erscheint
| Sleep | 5000 |

|script|Ldap Operation Script|ad|
|ensure | object |${scoped_ad_group}|exists|

Wird hingegen eine Gruppe in einem anderen als dem Gruppen Container erzeugt 

|script|Ldap Operation Script|metatree|
|create object|${unscoped_meta_group}|with class | groupOfNames| and attributes | !{description: Unscoped Group}|


wird diese Operation nicht übertragen

|script|Ldap Operation Script|ad|
|reject| object | ${unscoped_ad_group} | exists|
